## Submission Cadence & Strategy

Follow this schedule to maintain a natural link velocity:

### Week 1-2: High-Priority Submissions
- Product Hunt launch preparation and execution
- Crunchbase profile
- GitHub PR submissions (all 6 repos)
- Duuce (auto-submits to 4 directories)
- InboxReads

### Week 3-4: Mid-Priority Submissions
- Find Your Newsletter, Newsletry, Rad Letters, NewsletterHunt
- InboxStash, Thanks for Subscribing, Newsletters.co
- Paved publisher signup
- Reletter claim
- PostApex publisher signup

### Week 5-6: Lower Priority + Outreach
- LetterList, Letterstack, Email Love, Newsletterest, StackLetter
- G2 listing, AlternativeTo listing, Indie Hackers profile
- Begin listicle outreach (Tier 5 targets)

### Week 7-8: Follow-up & Verification
- Verify all listings are live and links are indexed
- Check Google Search Console for new backlinks appearing
- Follow up on pending GitHub PRs
- Follow up on listicle pitches that received no response

### Ongoing (Monthly)
- Search for new "best AI newsletters" or "best developer newsletters" listicles published each month
- Monitor backlink profile in Ahrefs/Semrush for lost links
- Submit to any new directories that emerge
- Track referral traffic from each directory in GA4

## Outreach Pitch Email Template

Use this template when reaching out to listicle authors, directory curators, or blog editors.

```
Subject: Spanish-language AI dev tools newsletter for your [article/directory] -- DevAI Semanal

Hi [Name],

I came across your [article title / directory name] and noticed it covers
[AI newsletters / developer newsletters / tech newsletters]. Great resource --
I have it bookmarked.

I wanted to suggest DevAI Semanal (https://devaisemanal.com) for inclusion.
It is a free weekly newsletter covering AI developer tools -- Claude Code,
Cursor, GitHub Copilot, MCP servers, and more -- written in Spanish.

A few reasons it might be a good fit:

- It fills an underserved niche: there are very few quality AI developer
  newsletters in Spanish, despite massive demand from Latin American and
  Spanish developers.
- Published weekly on Tuesdays with consistent, hands-on coverage of
  AI coding tools, prompt engineering techniques, and productivity workflows.
- Entirely free to subscribe -- no paywall, no premium tier.

If you think it would add value to your list, I would be grateful for the
mention. Happy to provide any additional details, screenshots, or a sample
issue.

Thanks for your time,
[Your name]
[Your email]
```

### Pitch Variations by Target

**For "Best AI Newsletters" listicles:**
Emphasize the Spanish-language angle and the AI coding tools niche (Claude Code, Cursor, Copilot). Mention that it is one of the few newsletters bridging the gap between English-language AI tool coverage and the Spanish-speaking developer community.

**For developer newsletter directories:**
Lead with the technical depth -- MCP servers, agentic coding workflows, model comparisons. Position it alongside established dev newsletters like TLDR and Bytes but for the Spanish-speaking segment.

**For GitHub awesome-lists:**
Keep the PR description short and factual. Follow the existing format exactly. Example entry:
```
- [DevAI Semanal](https://devaisemanal.com). A free weekly newsletter about AI developer tools (Claude Code, Cursor, Copilot, MCP) in Spanish.
```
